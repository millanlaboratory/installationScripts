## <span id="Loop"> Loop </span>
**OS tested: Ubuntu 14.04 LTS & 16.04 LTS (laptop & desktop), Ubuntu 18.04 LTS & 19.04 (desktop only)**
**Requirements:**
* python2.7
* libusb-1.0-0-dev
* git
* cifs-utils
* autoconf
* gnulib
* flex
* libfreetype6-dev
* libfontconfig1-dev
* libgstreamer0.10-0
* libxml2-dev
* libgstreamer-plugins-base0.10-dev
* libsdl1.2-dev
* libsdl-image1.2-dev
* libsdl-ttf2.0-dev
* libfreeimage-dev
* libgtk2.0-dev
* mwrap
* gcc-4.8
* MATLAB R2017b

**If libraries are not installed, the script will install them.**
